Normalised Data

Exiqon miRNA Microarrays - Array 1 of 2

Overview

The top two plots on this page are interactive "Shiny" objects. They're built using the actual data used in your analysis.

The profile plot gives you a visual representation of what happens in the normalisation process. Switching between Raw data and Normalised Data, you can see how the lines appear to straighten out. This is making probe intensities similar across arrays. The 'Percentile Range' is the number of probes included in the plot expressed as a percentage.

The PCA plot (Principle Component Analysis) Shows the variance between the data on an array level. This analysis is a very good method for detecting outliers. This plot has the ability for you to test the removal of different arrays and see how it influences clustering. - Warning - When each array is removed or added from the selection, the data must be re-normalised and this can take around 2 minutes.
